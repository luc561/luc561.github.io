Simon Say Via VHDL

View the Source Code On GitHub

A game that flashes a numerical pattern that is corresponds with a push button on a Xilinx Spartan 3 board. Programing the FPGA board to end the game with if the pattern is not matched exactly.

The FPGA is programed using states that only allow for one correct path. Winners receive a "uu" but if mistake is made an "XX" will appear forcing the user to reset the game to start over. Dip switch are used to identify levels and current difficulty. Three levels are available in this commit.
